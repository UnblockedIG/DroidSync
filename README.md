# NovaSync
NovaSync is a free web-based HTML Chromium ADB tool


How it works

NovaSync was made to work in chromium based browers (atleast most)

It uses a usb connection via usb debugging to access files and allow simple control and screen mirroring

It a simple sleek and well working web-based ADB tool

More improvments,Features and updates will come soon (Safari support next)
